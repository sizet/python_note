安裝 Python

在 Windows 7 x64 上安裝.

如果需要安裝 Python2 的話 :
1. 安裝設定.
安裝路徑設為 C:\Python2

2. 設定環境變數 Path.
在 Path 尾端加入 Python2 的安裝路徑 ";C:\Python2".


安裝 Python3 :
1. 安裝設定.
安裝時選擇 [Customize installation],
[Add Python to environment variables] 取消勾選,
安裝路徑設為 C:\Python3.

2. 設定環境變數 Path.
如果沒安裝 Python2 :
在 Path 尾端加入 Python3 的安裝路徑 ";C:\Python3;C:\Python3\Scripts".
如果有安裝 Python2, 需要 Python2/Python3 共存 :
Python2 的變數要在 Python3 的變數前面,
在 Path 尾端加入 ";C:\Python2;C:\Python3;C:\Python3\Scripts".


使用 pip 安裝套件的方法 :
Python2 :
py -2 -m pip install <package>
Python3 :
py -3 -m pip install <package>


執行 Python 程式的方法 :
Python2 的程式 :
py -2 <程式檔案路徑>
Python3 的程式 :
py -3 <程式檔案路徑>

或是在程式開頭指定使用 Python2 或 Python3 :
Python2 :
#!/usr/bin/env python2
Python3 :
#!/usr/bin/env python3

再使用 :
py <程式檔案路徑>

在 Xubuntu 16.04 x64 上安裝.

預設已經安裝好 Python-2.7 和 Python-3.5.

安裝 pip :
Python2 :
apt-get install python-pip
Python3 :
apt-get install python3-pip


使用 pip 安裝套件的方法 :
Python2 :
py -2 -m pip install <package>
Python3 :
py -3 -m pip install <package>


執行 Python 程式的方法 :
Python2 的程式 :
python2 <程式檔案路徑>
Python3 的程式 :
python3 <程式檔案路徑>
